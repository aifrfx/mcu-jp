# MCU JP Railway Ready

## Railway Variables

- `GEMINI_API_KEY`
- `GEMINI_PRIMARY_MODEL=gemini-3.5-flash`
- `GEMINI_FALLBACK_MODELS=gemini-3.1-flash-lite,gemini-2.5-flash-lite`
- Optional: `GEMINI_API_VERSION=v1beta`
- Optional: `GEMINI_TIMEOUT_MS=210000`

Do not set `PORT`. Railway supplies it automatically.

## Start command

`npm start`

## Healthcheck

`/health`
